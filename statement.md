# Problem Statement
Surfing internet for terminologies that are commonly used  in the field of finance to a user who has no/minimal knowledge about it is really brainstorming. This knowledge gap can lead to financial confusion, misinterpretation of reports and taxes, and suboptimal financial decision-making. The core problem is the lack of a simple, accessible tool for defining fundamental financial terminology in a concise manner. This code is made to solve this  problem and become one's financial dictionary.

# Scope of Variable
1. Variable Name: Terms
2. Function: It stores the text user types in and gives output accordingly.
3. Limitation: This code works only for these terms with respect to case sensitivity  - asset, liability, net worth, income, expenses, budget, cash flow, interest, compound interest, credit score, loan, equity, investment, diversification, return on investment,  bonds, balance sheet, capital gain, liquidity.
4. Does not work for: If term is like, education, then it shows that there will be an update soon.

# Targeted User
1. Beginners in Finance
2. Students
3. Someone in need to understand basic terms of finance

# High-Level Features
1. Term Based Diagonosis
2. Command Interface
3. Simple and easy to interpret definitions
4. Updation of new words
5. Instant response generation
