# FINANCE-GEEK-HUB

# Objective
The main objective of Finance Geek Hub is to act as a easy and interactive dictionary for fundamental financial and accounting terms. This tool is designed to help users to easily understand core concepts—such as assets, bonds, loan, etc—to educate people in financial literacy and reduce the risks that arise from misinterpreting these financial terminologies.

# Features
1. Definition Check: Accepts user input for many commonly used financial terms 
2. Command-line Interface: Simple text-based interaction for easy interpretation.
3. User Friendly: Simple and easy to execute.
4. Safety Fallback: identifies when a term isn't in the database and responds saying-'term will be updated soon'.

# Tools Used
1. Language: Python
2. Editor: Python IDLE
3. References: Modules and financial data from online sites

# Steps to Install and Run the Project
1. Install python
2. Open a new file in IDLE Shell
3. Create a filename Finance_Geek_Hub.py
4. Write code and save it
5. Execute code
6. Enter a term and get output

# Instructions for Testing
1. Start the program 
2. Enter the input term
3. If data is available it gives definition
4. If data is unknown it will respond that-'the term will be updated soon'
5. The program automatically ends after giving definition
6. For another term run it again
